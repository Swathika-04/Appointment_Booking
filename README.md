# Appointment Booking
#  Hospital Management System

A simple Hospital Management web application developed using the **MERN Stack** (MongoDB, Express.js, React.js, and Node.js). The application allows users to book hospital appointments and manage appointment records through an intuitive and responsive interface.

---

## 📌 Features

- Book hospital appointments
- Enter patient details
- Select doctor, date, and time
- Store appointment data in MongoDB
- View all booked appointments
- Responsive and user-friendly interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
